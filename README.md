# Sistema de Gerenciamento de Reservas de Hotel

# Descrição 

Este projeto consiste no desenvolvimento de um sistema de gerenciamento de reservas de hotel utilizando a linguagem Java, com base nos princípios da Programação Orientada a Objetos (POO).
A aplicação permite o cadastro, listagem, busca e ordenação de reservas, além de aplicar regras de negócio relacionadas ao cálculo do valor total da estadia, considerando o tipo de quarto selecionado.
O sistema foi projetado para simular um cenário real de controle de hospedagens, garantindo validações de entrada e organização eficiente dos dados.

# Objetivo

O objetivo principal deste projeto é aplicar, na prática, conceitos fundamentais de desenvolvimento em Java, tais como:

- Estrutura de classes e objetos
- Encapsulamento
- Validação de dados
- Manipulação de arrays
- Estruturas de repetição e decisão
- Implementação de algoritmos de ordenação
